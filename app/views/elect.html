Electoral System Simulator.

Chose values for the inputs then hit the execute button to run the election. Candidates and votes will be generated randomly based off the input values, the results will be calculated using the single transferable vote system and displayed below.

Inputs

  • Input the number of candidates who will stand at this election. All candidates are considered independent, party affiliations are not simulated.

  • Input the number of seats to be filled in this election.

  • Select the number of votes to cast.
